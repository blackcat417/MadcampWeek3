처음 기획한 기능들은 위와 같습니다. 하지만 3주차 진행과정에서 ‘ 반려식물 키우기 ‘ 기능만 구현을 완료.

추가로 식물 사진을 찍었을 때 이미지와 갤러리에서 가져온 이미지를 분석하여 식물 종을 알려주는 기능 추가.

### 1. 회원가입, 로그인

### 2. 내가 키우고 있는 반려 식물 등록하기

사진을 찍거나 갤러리의 이미지를 업로드하면 해당 이미지를 분석해 식물 종류를 알려준다.

결과가 정확하다면 ‘예’버튼을 누르면 사진이 반영이 되고 나머지 식물 종을 입력해 등록하면 Home 화면에서 나의 반려식물들 리스트에 추가된 것을 확인 할 수 있다.

### 3. 나의 반려식물 Gallery

등록된 반려식물마다 갤러리를 만들어 하루하루의 성장과정을 기록할 수 있다. 

### 4. Todo 달력

식물마다 물을 준다거나 영양제 등 해야할 일이 생기면 해당 날짜에 등록하여 달력에 기록할 수 있고,

완료된 할일은 체크 표시를 눌러 완료됨을 표시할 수 있다.
